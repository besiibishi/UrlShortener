# Url Shortener

A simple URL shortener project built with PHP, MySQL, HTML, and CSS for local use.

## Usage

1. Clone the repository.
2. Start MAMP.
3. Import the SQL database.
4. Configure database settings.
5. Access via `http://localhost/url-shortener`.
